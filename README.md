# ENVS-193DS_spring-2026_final
Repository representing cumulative final exam assignment featuring affective visualization, R data analysis , and research writing and interpretation skills.

This repository contains my final exam submission for ENVS 193DS. The project combines data visualization, statistical analysis, and scientific writing using bird observation data collected over 35 days. Analyses explore relationships between bird abundance, temperature, and weather conditions, with separate models developed for large and small bird counts.

## Data and file overview

bird_weather_observations copy.csv – Dataset containing bird observations collected during 20–30 minute walks to class or work over 35 days. Variables include date, observation duration, large bird counts, small bird counts, weather conditions, temperature, dew point, precipitation, and wind speed.

Final_Exam.qmd – Quarto document containing all code, analyses, visualizations, and written responses for the final exam.

Final_Exam.pdf – Rendered PDF version of the completed final exam report.

ENVS-193DS_spring-2026_final.Rproj – RStudio project file for the repository.
